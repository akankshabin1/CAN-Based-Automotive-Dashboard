# CAN-Based-Automotive-Dashboard
A multi-node embedded system project using CAN (Controller Area Network) to simulate an automotive dashboard with real-time data communication between multiple ECUs.

ECU3:
  PIC18F4580 microcontroller programming
  CAN protocol implementation and communication
  Embedded systems in C with XC8 compiler
  CLCD display interfacing and control
  Timer-based synchronization
  LED indicator control
  Multi-node embedded systems
